# Dabit 개인정보처리방침

이재관(이하 "운영자")은 정보주체의 자유와 권리 보호를 위해 「개인정보 보호법」 및 관계 법령이 정한 바를 준수하여, 적법하게 개인정보를 처리하고 안전하게 관리하고 있습니다.

본 개인정보처리방침은 운영자가 제공하는 모바일 애플리케이션 "Dabit"(이하 "서비스")의 이용과 관련하여 적용됩니다.

## 1. 수집하는 개인정보 항목

### 가. 필수 수집 항목
- **이메일 주소**: 회원가입, 본인 확인, 비밀번호 재설정
- **비밀번호**: 암호화하여 저장 (운영자도 원본 확인 불가)
- **서비스 이용 기록**: 습관, 토큰, 통계 데이터

### 나. 선택 수집 항목
- **닉네임**: 서비스 내 표시
- **프로필 이미지**: 서비스 내 표시

### 다. 자동 수집 항목
- **디바이스 정보**: OS 종류, OS 버전, 기기 모델
- **에러 로그**: 서비스 안정성 개선 (Sentry)
- **광고 식별자**: 향후 광고 서비스 도입 시 사용

## 2. 개인정보 수집 및 이용 목적
1. **회원 식별 및 인증**: 회원가입, 본인 확인, 부정 이용 방지
2. **서비스 제공**: 습관 관리, 독서 기록, 공부 시간 측정 등
3. **서비스 개선**: 에러 추적, 사용 패턴 분석 (익명화)
4. **고객 지원**: 문의 응답
5. **향후 광고 서비스**: 광고 도입 시 별도 동의 후 사용

## 3. 개인정보 보유 및 이용 기간

운영자는 회원이 서비스를 이용하는 동안 개인정보를 보유합니다.

### 가. 회원 가입 기간
- 회원이 서비스를 이용하는 기간 동안 보유

### 나. 회원 탈퇴 시
- 즉시 모든 개인정보 및 활동 기록 삭제
- 삭제된 정보는 복구 불가

### 다. 법령에 따른 보관
관계 법령에 따라 일정 기간 보관이 필요한 경우 해당 기간 동안 보관:
- 「전자상거래법」 등에 따른 거래 기록 보관 (해당 시)

## 4. 개인정보의 제3자 제공

운영자는 정보주체의 동의 없이 개인정보를 제3자에게 제공하지 않습니다.

다만, 다음의 경우는 예외로 합니다:
- 정보주체로부터 별도 동의를 받은 경우
- 법령에 의해 요구되는 경우
- 통계작성 등을 위해 특정 개인을 식별할 수 없는 형태로 가공하여 제공하는 경우

## 5. 개인정보의 처리 위탁

운영자는 다음과 같이 개인정보 처리 업무를 위탁하고 있습니다:

| 위탁받는 자 | 위탁 업무 | 보유 기간 |
|---|---|---|
| Supabase Inc. | 데이터 저장 및 관리 (클라우드) | 회원 탈퇴 시까지 |
| Functional Software, Inc. (Sentry) | 에러 추적 및 분석 | 90일 |
| Kakao Corp. | 책 정보 검색 API | 검색어만 일시 사용 |
| Google LLC (AdMob) | 광고 서비스 (향후 도입 시) | 광고 표시 기간 |

## 6. 정보주체의 권리

회원은 다음과 같은 권리를 행사할 수 있습니다:

1. **개인정보 열람**: 앱 내 설정 페이지에서 확인 가능
2. **개인정보 수정**: 앱 내 설정 페이지에서 직접 수정
3. **개인정보 삭제**: 회원 탈퇴를 통해 모든 정보 삭제
4. **처리 정지 요구**: 이메일을 통해 요청

권리 행사는 앱 내 설정 페이지 또는 다음 이메일을 통해 가능합니다:
**dabit.help@gmail.com**

## 7. 개인정보 보호를 위한 기술적·관리적 조치

### 가. 기술적 조치
- 비밀번호 암호화 (bcrypt)
- HTTPS 통신 (TLS 1.2 이상)
- 데이터베이스 접근 권한 최소화
- 정기적인 보안 점검

### 나. 관리적 조치
- 개인정보 취급자 최소화 (운영자 본인)
- 개인정보 보호 교육 및 점검

## 8. 광고 식별자에 관한 사항

### 가. 수집하는 광고 식별자
- Android 광고 식별자 (AAID)
- iOS 광고 식별자 (IDFA)

### 나. 수집 시점 및 목적
- **현재**: 광고 식별자를 수집하지 않습니다.
- **향후 도입 시**: 사용자 별도 동의 후 광고 표시 및 측정 목적으로 사용

### 다. 거부 방법
- Android: 설정 > Google > 광고 > 광고 ID 재설정/삭제
- iOS: 설정 > 개인정보 보호 > 추적 > 앱이 추적 요청 허용

## 9. 만 14세 미만 아동의 개인정보

운영자는 만 14세 미만 아동의 회원가입을 받지 않습니다.

만약 만 14세 미만 아동이 가입한 사실이 확인되면 즉시 회원자격을 박탈하고 모든 정보를 삭제합니다.
